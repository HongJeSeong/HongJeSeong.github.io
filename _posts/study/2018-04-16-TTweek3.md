## 1.
첫번째 배열에서 가장 작은 값, 두번째 배열에서 가장 큰 값을 뽑고 이들을 배열에서 제거한다. 뽑은 값을 곱하여 더하기를 반복한다.  

```python
def getMinSum(A,B):
    answer = 0
    size=len(A)
    while size > 0:
        num1=min(A)
        A.remove(num1)
        num2=max(B)
        B.remove(num2)
        size-=1
        answer+=num1*num2
    return answer

#아래 코드는 출력을 위한 테스트 코드입니다.

print(getMinSum([1,2],[3,4]))
```
